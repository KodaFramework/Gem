Commands

> help
Will show you a list of commands, can be called at any time.
> ls
List current directory contents
> cd [directory]
Navigate to a directory
> mkdir [resource]
Creates a folder
> open [resource]
Views a resource
> peek [resource]
Shows the raw content of a resource
> edit [resource]
Edits or Creates a resource in a folder
> rm [resource]
Deletes a resource.
> tab completion
Press tab when navigating to complete
> Use & for multiple commands
example 'help & cd articles & ls'
